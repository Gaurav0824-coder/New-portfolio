# Styled Portfolio Website

## Project Overview

This project is a personal portfolio website built using HTML5 and CSS3.  
It showcases my understanding of web structure, styling, layout techniques, and responsive design.

## HTML Concepts Learned

- Semantic HTML tags (`header`, `nav`, `section`, `main`, `footer`)
- Internal navigation using anchor links
- Forms with input validation
- Image usage with alt attributes
- Clean document structure

## CSS Concepts Learned

- External CSS file linking
- CSS selectors (element, class, pseudo-class)
- Flexbox and Grid for layout
- Hover effects and transitions
- Responsive design using media queries
- Custom fonts and color schemes

## Folder Structure

- index.html – Main HTML file
- style.css – Styling and responsive design
- images/ – Profile image
- README.md – Documentation
